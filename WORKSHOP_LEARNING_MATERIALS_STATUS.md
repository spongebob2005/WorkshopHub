## 🔧 Customization Examples

### Add a New Workshop
```json
{
  "title": "Your Workshop Title",
  "instructor": "Your Name",
  "price": 99.99,
  "tutorials": [
    "Module 1: Title - Description",
    "Module 2: Title - Description",
    // ... 8 modules
  ],
  "learningContent": {
    "mcqTests": [
      {
        "id": "q1",
        "question": "Your question?",
        "options": ["A", "B", "C", "D"],
        "answer": "Correct answer",
        "explanation": "Why... detailed explanation"
      }
      // ... 6+ questions
    ],
    "videoResources": [/* 5 videos */],
    "pdfResources": [/* 5 PDFs */]
  }
}
```

Then run: `node backend/seed.js`

### Update Existing Content
- Edit `backend/seed-workshops.json`
- Modify titles, descriptions, modules, questions, resources
- Run seed script: `node seed.js`
- Changes appear immediately!
